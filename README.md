# Face Attendance System v4.1 - Manual-only (Timestamped saves)

This version provides:
- Manual attendance (no timetable). Click 'Take Attendance' and enter a subject to start.
- Attendance saved with a timestamp: Subject_YYYY-MM-DD_HH-MM-SS.xlsx (prevents overwriting).
- Admin Panel for adding students, managing student DB, training model and viewing attendance.
- Multi-angle capture for registration.
- Attendance auto-captures for CONFIGURED duration (default 20s) and saves as Excel (.xlsx).

Setup:
1. pip install -r requirements.txt
2. Put real haarcascade_frontalface_default.xml in project root.
3. Run: python main.py
